# IoT Sunrise Alarm

## Installing
Create a virtualenv and activate it:

```
$ python3 -m venv .venv
$ . .venv/bin/activate
```

Or on Windows cmd:

```
$ py -3 -m venv .venv
$ .venv\Scripts\activate.bat
$ .\.venv\Scripts\Activate.ps1
```

Install Flaskr:

```
$ pip install -e .
```

## Run
```
$ flask --app flaskr run --debug
```
Open http://127.0.0.1:5000 in a browser.
